# Rapidly Exploring Random Trees (RRT)
The Rapidly Exploring Random Trees (RRT) algorithm serves as a path planning tool in navigating from a starting point (洧논<sub>洧멇롐뫯롐뀛롐洧노</sub>) to a destination (洧논<sub>洧녬洧뉧롐멇롐</sub>) within a defined metric space (洧녦). This algorithm operates by searching through the free space (洧녦<sub>洧녭洧洧뉧롐</sub>), avoiding obstacles (洧녦<sub>洧녶洧녪洧</sub>). The process begins by randomly sampling a point (洧논<sub>洧洧녩洧녵洧녬</sub>) and finding the nearest neighboring point (洧논<sub>洧녵洧뉧롐뀛롐</sub>) using Euclidean distance. A tree is then constructed, expanding from 洧논<sub>洧멇롐뫯롐뀛롐洧노</sub> to 洧논<sub>洧녵洧뉧롐뀛롐</sub>.

The algorithm continues by connecting 洧논<sub>洧녵洧뉧롐뀛롐</sub> to a newly generated point (洧논<sub>洧녵洧뉧롐</sub>) and checking for collisions with obstacles. If successful, 洧논<sub>洧녵洧뉧롐</sub> is added to the tree, and the process repeats until reaching the destination or reaching a specified iteration limit. However, due to the random nature of the algorithm, the path may contain redundant turning points, resulting in inefficiency.







## Overview of script

This Python script implements Rapidly Exploring Random Trees (RRT) for path planning in a 2D space with obstacles. The main file, 'rrt_path_planning.py', generates an RRT, visualizes the exploration process, and outputs the final path. The script is compatible with Ubuntu 20. For proper execution, ensure the required Python library is installed.

## Requirements

- Python 3
- Matplotlib

## Installation

Install the required libraries using the following command:

    sudo apt install python3

    pip install matplotlib
